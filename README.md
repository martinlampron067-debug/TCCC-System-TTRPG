# TCCC SYSTEM: Tactical Combat & Creative Cinematic System
### Realistic RPG Toolkit | Fast-Play Precision | Veteran-Designed

The **TCCC System** is a high-stakes, realistic tabletop role-playing engine designed for tactical depth and cinematic storytelling. Created by Martin Lampron (Canadian Armed Forces Veteran), this system moves away from traditional "Rule-Lite" mechanics toward a **Realistic Toolkit** that remains fast enough for fluid play.

## 🛠️ Core Mechanics
* **Scalable Dice Resolution:** No d20s. Attributes and Skills use a d4 to d12 scale, creating a predictable Bell Curve that favors player competence over random luck.
* **Injury Over Hit Points:** Traditional health bars are eliminated. Track realistic impacts from **Stun to Lethal** that directly affect character effectiveness in real-time.
* **Genre-Agnostic Engine:** A universal toolkit designed to cover all ages of human evolution—from primitive survival to high-tech space opera.

## 📚 Current Modules & Expansions
### [Saeculum: Blood of Ages](https://martinlampron067-debug.github.io/TCCC-System-TTRPG/)
The core engine for the TCCC System. Designed for modular, multi-genre play where survival depends on tactical safety, not hit point attrition.

### [Saeculum: Carneferreus](https://martinlampron067-debug.github.io/TCCC-System-TTRPG/)
A cyberpunk expansion set in a fractured, post-apocalyptic North America. Features gritty "Data Hustle" cracking and "Chrome Sickness" augmentation costs.

### [TCCC Lite: The Blackjack System](https://www.drivethrurpg.com/en/publisher/33737/martin-lampron)
An over-simplified, zero-prep Western version of the TCCC system. Perfect for filler sessions or handling late players with rapid blackjack-based resolution.
